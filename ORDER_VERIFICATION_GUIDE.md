# 永豐證券訂單驗證指南

## 📋 訂單狀態說明

### Line 回馈的「成功」代表什麼？

**「限價買單成功」= 訂單已送出到永豐證券**
- ✅ API 調用成功
- ✅ 訂單已進入委託系統
- ❌ **不代表立即成交**
- ❌ **需等待市價達到限價才會成交**

### 訂單生命週期

```
[1] Line下單 
  ↓
[2] 訂單送出 ← Line回馈「成功」（此階段）
  ↓
[3] 委託中 (等待成交)
  ↓
[4] 部分成交 / 完全成交 / 已取消
```

## ✅ 4種驗證方法

### 方法 1：Line 查詢訂單狀態（新功能）

**查詢所有委託單**：
```

```

**查詢特定股票**：
```
查單 2330
```

**顯示內容**：
```
📋 委託單列表 (3 筆):
✅ 2330 | BUY 100股 @ $580 | 完全成交
⏳ 2330 | BUY 50股 @ $575 | 委託中
❌ 2454 | SELL 200股 @ $120 | 已取消
```

**狀態說明**：
- ✅ **完全成交**：訂單已全部成交
- ⏳ **委託中**：等待價格達到限價
- ❌ **已取消**：訂單已取消或失效

### 方法 3：永豐證券交易軟體

**Web 版或手機 APP**：
1. 登入永豐證券交易系統
2. 查看「**委託查詢**」→ 確認訂單存在
3. 查看「**成交查詢**」→ 確認是否成交

**委託狀態對照**：
- **已報（Submitted）**：訂單已送交交易所
- **部分成交（PartiallyFilled）**：部分股數已成交
- **完全成交（Filled）**：全部股數已成交
- **已刪（Cancelled）**：訂單已取消

## ⚙️ 進階：訂單追蹤

### 如需即時通知成交

可設定永豐證券的**回報機制**（需修改代碼）：

```python
# 在 shioaji_handler.py 中添加
def setup_order_callback(self):
    @self.api.on_order_update
    def order_callback(trade):
        if trade.status.status == "Filled":
            # 發送 Line 通知
            send_line_notification(f"✅ {trade.contract.symbol} 已成交！")
```
