# 台股交易策略配置指南

## ✅ 已完成配置

台股現已支持**限價買入 + 自動賣出**策略，類似美股的 Bracket Order：

- 🎯 **限價買入**：低於現價掛買單
- 📈 **固定獲利賣出**（獲利目標 < 5%）：限價賣單
- 📉 **追蹤止損賣出**（獲利目標 ≥ 5%）：買單成交後需手動設置

---

## 🎯 使用方式

### 方式 1：固定獲利賣出（默認）

**Line 指令**：
```
買 2330 100
```

**系統行為**：
1. 限價買入：現價 × 98.5% = $573（假設現價 $580）
2. 限價賣出：買入價 × 103% = $590

**特點**：
- ✅ 同時掛兩張單
- ✅ 買進後自動掛出賣單
- ⚠️ 賣單為固定價格，不會追蹤

---

### 方式 2：追蹤止損（高獲利目標）

**Line 指令**：
```
買 2330 100 10%   # 獲利目標 10%
```

**系統行為**：
1. 限價買入：現價 × 98.5% = $573
2. ⚠️ **追蹤止損需手動設置**（Shioaji 限制）

**Line 回餽**：
```
🚀 台股 限價買單已送出！
📌 代號：2330 (數量:100)
🔹 買入限價：$573
⚠️ 追蹤止損需在成交後手動設置
```

---

## 🔧 自動選擇邏輯

系統會根據獲利目標自動選擇策略：

| 獲利目標 | 使用策略 | 說明 |
|---------|---------|------|
| < 5% | 固定限價賣出 | 自動掛雙單 |
| ≥ 5% | 追蹤止損 | 需手動設置 |
| 未指定 | 固定限價賣出 | 默認 3% |

**範例**：
```bash
買 2330 100        # 固定獲利 3%（默認）
買 2330 100 3%     # 固定獲利 3%
買 2330 100 8%     # 追蹤止損 8%（手動）
```

## ⚠️ 永豐證券（Shioaji）限制

### 1. 無自動追蹤止損

**Why？**
- Shioaji SDK 不支持類似 IB 的自動 bracket order
- 追蹤止損需要在買單成交後手動送出

**解決方案**：
1. **監聽成交回報**（進階功能，需開發）
2. **手動設置**：成交後發送 Line 指令

---

### 2. 固定限價賣出的風險

**情況**：
- 買入價：$573
- 賣出價：$590（固定）
- 股價大漲到 $650

**結果**：
- ❌ 在 $590 就賣掉了
- 😢 錯過後面 $60 的獲利

**建議**：
- 獲利目標較大時（>5%），使用追蹤止損
- 手動調整賣單價格

## 🔍 常見問題

### Q1：為什麼不使用市價買入？

**A**：台股市場波動較大，使用限價買入可以：
- ✅ 避免追高
- ✅ 控制買入成本
- ✅ 等待回調進場

### Q2：如何手動設置追蹤止損？

**A**：需要等買單成交後，在永豐交易軟體中手動設置移動停利單。

### Q3：買賣單會互相沖突嗎？

**A**：不會。
- 買單未成交：賣單不會執行（沒有持倉）
- 買單成交：賣單等待目標價
